Vyper-Cloud™ SDK

The Vyper-Cloud™ SDK is a Software Development Kit that makes building and deploying a Computing Cloud Fast and Easy.

The Vyper-Cloud™ SDK Features

Contact us to build your next Computing Cloud.
  • Fully compatible with Linux and Windows OS's
    • Any Node in a Vyper-Cloud™ can use any version of Linux or any version of Windows.
    • Vyper-Cloud™ Nodes are dynamic, Nodes can be created and destroyed in a fully dynamic manner..
  • Vyper-Cloud™ supports an elastic model; all Nodes are fully dynamic using any TCP/IP address that is accessible to the other Nodes in the Cloud.
  • Auto-Discovery of Vyper-Cloud™ Nodes. Whenever a node initializes it becomes avalable to the other Nodes in the Cloud.
  • Vyper-Cloud™ Nodes can span Networks. Nodes can be configured to forward requests from one Network to another Network.
  • Multiple Nodes can be configured on the same physical computer to maximize computing resources and computing cores.
  • Vyper-Cloud™ Nodes can be configured to execute any program or service that can be run on the computer that hosts the Node.
  • Vyper-Cloud™ can manage persistent data in a distributed manner using any available Relational or non-Relational database such as mySQL, SQL Server, Oracle or our own proprietary database, Vyper-Stor™.
  • Vyper-Cloud™ Nodes use SSL or SSH to achieve Secure connections for each Node.
  • Vyper-Cloud™ provides a Secure Console to facilitate Administration of the Cloud.
