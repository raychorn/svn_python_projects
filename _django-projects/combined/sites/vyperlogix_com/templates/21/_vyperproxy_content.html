Vyper-Proxy™ Enterprise Edition could be used for the following:
  • Accelerate your current web server to serve content as much as 2x faster or more.
    • A very small configuration of Vyper-Proxy™ Enterprise Edition was benchmarked serving content from Apache 2.2.8 Load Balancer configured as follows:
      • This same configuration was able to serve content as fast as Apache 2.2.8 using Load Balancer serving requests through 3 Django Instances.
        • Apache 2.2.8 Load Balancer serving requests through 20 Vyper-Proxies.
        • Vyper-Proxy serving content through 3 Django Instances.
  • Replace your current web server with Vyper-Proxy™ Enterprise Edition.
    • Vyper-Proxy™ Enterprise Edition serve requests through many persistent Django Application Instances.
    • Vyper-Proxy™ Enterprise Edition can span multiple physical servers each with many persistent Django Application Instances.
    • Vyper-Proxy™ Enterprise Edition can help you build your own Django Cloud.
    • Vyper-Proxy™ Enterprise Edition can serve request to any HTTP/1.1 Application including but not limited to any of the following:
      • Python+Django (any version of either)
      • Python+TurboGears (any version of either)
      • Python+WSGI (any version of either)
      • PHP (any version)
      • Java (any version)
      • .Net Applications (any version, any application)
      • IIS Applications (any version, any application)
      • Apache (any version, any application)
      • LiteSpeed (any version, any application)
      • Any HTTP/1.1 Web Server (any version, any application)
    • Each Vyper-Proxy™ Enterprise Edition "node" can service requests for an arbitrary list of HTTP/1.1 End-Points.
      • Each Vyper-Proxy™ "node" is an HTTP/1.1 End-Point
      • Any Program or Application that services HTTP/1.1 Requests is also an HTTP/1.1 End-Point
    • Vyper-Proxy™ Enterprise Edition "nodes" be networked to cause HTTP/1.1 Requests to be serviced faster.
      • Cascading downward networks can be more efficient than linear uni-dimensional networks.
      • Networking many physical computers together as follows can be very powerful:
        • Each physical computer gets a large Cascading downward network of Vyper-Proxy™ Enterprise Edition "nodes".
        • The bottom leaf of each node tree is a Vyper-Proxy™ Enterprise Edition "node" that pushes requests to another distributed Vyper-Proxy™ Enterprise Edition "node" network on another computer.
