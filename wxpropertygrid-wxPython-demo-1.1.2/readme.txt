PROPGRID_DEMO: wxPropertyGrid demonstration program, wxPython edition.
January 1, 2008.


LICENSE
PROPGRID_DEMO is licensed under the terms of the wxWidgets license.


PURPOSE
PROPGRID_DEMO is a wxPython clone of the c++ version of the wxPropertyGrid sample program.
The intended purpose is to provide sample code as an aid to learning how to use wxPropertyGrid.


CREDITS
PROPGRID_DEMO was written by Loren Jones.
