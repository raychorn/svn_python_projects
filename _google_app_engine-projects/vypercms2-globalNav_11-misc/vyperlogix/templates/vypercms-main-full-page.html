VyperCMS V2.0

High-powered easy to use and easier to deploy Python-based Content Management System running on the Google App Engine and Django 1.1.1.

VyperCMS V2.0 Features:

  • Database driven - all content can be stored and maintained online in the Google Datastore via the Django Admin Interface.
  • Template driven - content can be stored and maintained in standard Django Templates.
  • Flexible - Pages can be reconfigured through the use of easy to manage templates.
  • Dynamic - content can be entered by users or generated automatically from external data sources via RSS or other data feeds.
  • Scalable - Google App Engine provides elastic demand-driven scalability.
  • Compatible - works with Django 1.1.1 - can be deployed using Google App Engine or natively using a LAMP-based server.
  • Extensible - allows extensions to be added by anyone who can work with Django 1.1.1.
    • Works with jQuery, Dojo, Yahoo! User Interface (YUI), Ext JS, Mootools, Prototype, Scriptaculous, Adobe Spry, Microsoft ASP.NET Ajax Controls, MochiKit, Aflax and others...
    • Works with Flash and Flex - includes a powerful swfObject interface with Dynamic Embedding Support.
    • Works with Silverlight.
    • Support AJAX.

VyperCMS V2.0 Use Cases

VyperCMS V2.0 works best in an environment where non-programmers are called upon to maintain their content. Non-programmers are typically those who don't have any idea what HTML is all about nor do they know anything about Python or Django.
