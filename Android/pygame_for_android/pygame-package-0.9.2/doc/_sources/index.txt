The Pygame Subset for Android is a port of a subset of
`Pygame <http://www.pygame.org>`_ functionality to the
`Android <http://www.android.org>`_ platform. The goal of the
project is to allow the creation of Android-specific games, and
to ease the porting of games from PC-like platforms to Android.

Downloads
---------

The Pygame Subset for Android is `available from the Android Market <http://market.android.com/details?id=org.renpy.pygame>`_

Other Downloads:

* Packaging Tool: http://pygame.renpy.org/dl/pygame-package-0.9.2.tar.bz2
* APK Files: http://pygame.renpy.org/dl/
* Source Code & Bug Tracker: https://launchpad.net/pgs4a

Support
-------

The best place to get Pygame Subset for Android support is `our forum <http://pygame.renpy.org/forum/>`_.
   
Samples
--------

To use these, create a pygame directory on your device's SD card, and
then unzip them there. Source code is included.

`Slidepuzzle <http://pygame.renpy.org/dl/slidepuzzle.zip>`_

    Written by Al Sweigert, and found `here
    <http://inventwithpython.com/blog/category/code-comments/>`_, this
    is the classic sliding numbers puzzle.
